Predictive Analysis¶

In this notebook, we read in our previously processed data and perform the core of our machine learning analysis. After further cleaning the data, we attempt to predict (categorically) if a flight is delayed. Our main method is Random Forests, which we tune and fit over a very large 1,000,000 flight random sample of our data. As in our other parts, this process gives us an estimate of our out of bag error. We also tried as an extension to summarize the data through a 2-dimensional svd and perform a k nearest neighbors analysis. As a last extension, we attempt an svm classifier with a much smaller sample of the data, as the run time increases severly with the number of data points. We ran into difficulties with the SVM classification, but include the code to show our work. As expected, our highly trained, flexible random forest classifier performed strongest. We note that our metric of performance was the f1 score, which balances concerns with precision and recall.

Random Forest¶

In this section we write functions to tune and fit a random forest classifier. We do this over a large random subset of our data and find good performance relative to what was observed in previous experiments (cited earlier in our writeup). In particular, we expect that if were to scale to the size at which the cited experiments were conducted, we would have similar results with respect to f1 score.

#A function that takes a test dataframe and a trained RF classifier and
#returns the Precision, Recall, and F1 score

def RandFor_eval(RFfitted,Data,c,Y,X=None):
    #Tartet Data
    y=Data[Y]
    y=np.array(Data[Y])
    
    if X==None:
        predictedy=RandFor_cut(RFfitted,Data,c,Y)
        
    else:
        predictedy=RandFor_cut(RFfitted,Data,c,Y,X)

    #We will assume that categories are binary.
    #We define precision and recall through true positives and false positives
    tp=0.
    fp=0.
    for i in range(len(predictedy)):
        if(y[i]==1 and predictedy[i]==1):
            tp+=1
        elif(y[i]==0 and predictedy[i]==1):
            fp+=1
    recall=tp/float(np.sum(y))
    precision=tp/(tp+fp)
    f1score=(2*recall*precision)/(recall+precision)
    
    accuracy=0.
    for i in range(len(predictedy)):
        if(y[i]==predictedy[i]):
            accuracy+=1
    accuracy=accuracy/len(predictedy)
    
    return recall, precision, accuracy, f1score

#Similar to our RandForest function. Takes the same inputs but tunes the cutoff and returns this cutoff
def Findcutoff(Data,Y,optimaltrees,X=None):
    #Tartet Data
    y=np.array(Data[Y])
    
    #Case in which no feature list is specified...features are taken to be all
    #non-target columns.
    if X==None:
        #We avoid changing the Data matrix itself and deep copy
        xframe=pd.DataFrame()
        xframe=Data[Data.columns]
        xframe.drop([Y],axis=1,inplace=True)
        x=xframe.as_matrix()
        features=xframe.columns
        
    else:
        xframe=pd.DataFrame()
        xframe=Data[X]
        x=xframe.as_matrix()
        features=X
   
    #We take the best peforming number of trees and then use this to train our final classifier.
    RFCfinal=RandomForestClassifier(optimaltrees, n_jobs=-1)
    
    #We now parameter tune the optimal cut-off in predictions
    #We will return the optimal cutoff, so that it may be used in the application of the fitted tree
    def cutoff_predict(trained,DATA,cutoff):
        probabilities2temp=trained.predict_proba(DATA)
        predicttest2=[1 if x[1] > cutoff else 0 for x in probabilities2temp]
        return(predicttest2)

    def custom_f1(cutoff):
        def f1_cutoff(clf, X, y):
            ypred = cutoff_predict(clf, X, cutoff)
            return sklearn.metrics.f1_score(y, ypred) 
        return f1_cutoff

    #We use 10-fold cross validation to find the best cutoff value
    nvals2=np.arange(0.1,0.9,0.1)
    scorescutoff=[]
    cutoffmeanscores=[]
    
    for i in nvals2:
         f1_cut=custom_f1(i)
         scored=cross_val_score(estimator=RFCfinal,X=x,y=y,cv=10, scoring=f1_cut)
         scorescutoff.append(scored)
         cutoffmeanscores.append(np.mean(scored))
    
    cutoff_opt = nvals2[np.argmax(cutoffmeanscores)]     
        
    return cutoff_opt

#Function to parameter tune and fit an SVM classifier
def SVMtunedfit(Data,Y,X=None):
    #Tartet Data
    y=np.array(Data[Y])
       
    params={'C': [.1,100,10000], 'gamma': [.1,1,100]}
    #params={'C': [1,1000], 'gamma': [0,1]}
    
    #Case in which no feature list is specified...features are taken to be all
    #non-target columns.
    if X==None:
        #We avoid changing the Data matrix itself and deep copy
        xframe=pd.DataFrame()
        xframe=Data[Data.columns]
        xframe.drop([Y],axis=1,inplace=True)
        x=xframe.as_matrix()
        features=xframe.columns
        
    else:
        xframe=pd.Dataframe()
        xframe=Data[X]
        x=xframe.as_matrix()
        features=X

    #We parameter tune C and gamma, using the f1 scoring, to balance desire
    #for strong precision and recall
    svc=sklearn.svm.SVC()
    opt = sklearn.grid_search.GridSearchCV(svc, params, cv=10, scoring='f1', n_jobs=-1)
    
    opt.fit(x,y)
     
    
    #We return the optimized classifier as well as the optimal parameters found
    #and the grid scores, which can be used to visualize the optimization procedure
    #if so desired.
    return opt.best_estimator_, opt.best_params_, opt.grid_scores_

#SVD function that takes the data, a dimension, and the target column name as well as an optional list
#of covariate column names and outputs an svd matrix of the specified dimension.
#it also plots the given data on the principle components if the specified number of dimensions is 2
def svddat(Data,dim,Y,X=None):
    #Tartet Data
    y=np.array(Data[Y])
    
    #Case in which no feature list is specified...features are taken to be all
    #non-target columns.
    if X==None:
        #We avoid changing the Data matrix itself and deep copy
        xframe=pd.DataFrame()
        xframe=Data[Data.columns]
        xframe.drop([Y],axis=1,inplace=True)
        x=xframe.as_matrix()
        features=xframe.columns
        
    else:
        xframe=pd.Dataframe()
        xframe=Data[X]
        x=xframe.as_matrix()
        features=X
    
    #We compute and plot the first two principal components
    svd = sklearn.decomposition.TruncatedSVD(n_components=2)
    x_d = svd.fit_transform(x)

    if dim == 2:
        plt.scatter(x_d[:,0], x_d[:,1], c=y, s = 50, cmap=plt.cm.Paired)
        plt.colorbar()
        plt.xlabel('First Component')
        plt.ylabel('Second Component')
        plt.title('2-dim PCA Projection of Training Set')
        plt.show()
    
    return x_d

#Function to tune and fit a knn classifier, assuming the data is already in
#matrix form due to pca decomposition function above
#We will return the fitted classifier and the number of neighbors we found optimal
#We will also plot the scores
def knntuned(x,y):
    #We parameter tune the number of neighbors, using the f1 scoring, to balance desire
    #for strong precision and recall
    nvals=[1,3,5,7,10,20]
    
    scores=[]
    meanscores=[]
    for i in nvals:
        knn = sklearn.neighbors.KNeighborsClassifier(i)
        scored=cross_val_score(estimator=knn,X=x,y=y,cv=10, scoring='f1')
        scores.append(scored)
        meanscores.append(np.mean(scored))
        
    #We take the best peforming number of trees and then use this to train our final classifier.
    knnfinal=sklearn.neighbors.KNeighborsClassifier(nvals[np.argmax(meanscores)])
    knnfitted=knnfinal.fit(x,y)
    
    #We plot the scores in a histogram to visualize the results of parameter tuning
    scoredata=np.array(scores).T
    plt.figure(figsize=(10,7))
    plt.xlabel("Number of Neighbors")
    plt.ylabel("F1 Scores")
    plt.title("Cross Validation Score vs # Neighbors in KNN")
    sns.boxplot(scoredata, names=(nvals))
    plt.show()
  
    return knnfitted, nvals[np.argmax(meanscores)]
